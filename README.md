# Personal Website

Welcome to my personal website! This site features a sleek glassy effect and includes several useful features. Feel free to use this project freely without any limitations.

## Features
- **Glassy Effect Design**: A clean and elegant background with a translucent glass effect for a modern look.
- **Chat Feature**: An interactive chat option where visitors can communicate.
- **Admin Dashboard**: A simple admin dashboard where the owner can log in and log out. (Note: Currently no database connection, all static data).
- **Contact Form**: A form for visitors to send an email or contact the website owner directly.

## Installation

To get started, follow these simple steps:

1. **Clone or Download** this repository to your local machine:
   ```bash
   git clone https://github.com/yourusername/personal-website.git
````

2. **Open the project** in your preferred code editor.

3. **Preview the website**: Open `index.html` in your browser to see the site live.

## How to Use

* **Chat Feature**:

  * The chat feature allows visitors to interact directly through the website. Please note this is a static chat interface and does not currently store conversations.

* **Admin Dashboard**:

  * To access the admin dashboard, you can log in via the login screen. The dashboard allows the website owner to log in and out (currently static with no database).
  * Use the default login credentials to access the admin panel (this can be updated in the code).

* **Contact Form**:

  * Visitors can fill out the form on the contact page to send an email directly to the website owner. Currently, this is a static form and does not have backend functionality yet.
